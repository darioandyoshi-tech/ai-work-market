# Revenue Prospect Shortlist

High-signal public targets for AI Work Market paid/custom integration outreach. No one has been contacted yet.

## Recommended first wave

### 1. daydreams / Lucid Agents

- Link: https://github.com/daydreamsai/lucid-agents
- Fit: TypeScript framework for monetized agents with x402, A2A, ERC-8004, and payment policies.
- Likely pain: Needs credible services/work supply for agent-commerce examples and monetization flows.
- Public channel: GitHub Issues; org social listed publicly as `@daydreamsagents`.
- Suggested opener:

> Lucid already handles agent payments + policies; AI Work Market could be a live paid-work endpoint agents can discover/use in examples. Want a small adapter/demo PR?

### 2. Agent-Bazaar

- Link: https://github.com/Agent-Bazaar/Agent-Bazaar
- Fit: Autonomous AI-agent marketplace on Solana with SDK + MCP for discovery, hiring, payments, and reputation.
- Likely pain: Marketplace liquidity, useful paid jobs, and cross-chain/Base/x402 interoperability.
- Public channel: GitHub Issues.
- Suggested opener:

> Agent-Bazaar has the marketplace primitives; AI Work Market could supply real paid human/custom-integration tasks as a category. Worth exploring a bridge or listed service?

### 3. PayanAgent

- Links: https://payanagent.com / https://github.com/derNif/payanagent
- Fit: Open-source x402/Base marketplace where agents discover, hire, and pay agents/SaaS services.
- Likely pain: Differentiated service supply and trustable fulfillment.
- Public channel: GitHub Issues.
- Suggested opener:

> PayanAgent’s request marketplace maps closely to custom integration work. AI Work Market could be an early service/provider with escrowed delivery examples.

### 4. AetherCore ag402

- Link: https://github.com/AetherCore-Dev/ag402
- Fit: “Zero code change” x402 payments for OpenClaw, Claude Code, Cursor, LangChain, CrewAI, AutoGen.
- Likely pain: Strong examples of paid agent work, not just paid API calls.
- Public channel: GitHub Issues.
- Suggested opener:

> ag402 could make AI Work Market callable from coding agents with minimal setup — agent discovers a task, pays, and receives integration output.
