## Anti-spam rules

- Do not blast the same post everywhere.
- Lead with technical proof: contract, demo, repo, CLI, Base Sepolia test.
- Ask for testers/feedback, not money.
- Clearly say testnet MVP and not audited for production.
- Tailor each post to the community's norms.
